# Hadith-Watch-WearOS
This is a smartwatch application built for WearOS. It is designed to display a Hadith (quote) to users. 
It contains a refresh button to allow a new hadith to be displayed.

## Built With

* Kotlin
* Jetpack Compose
* Built in Android Studio
